Drop your SRH-2D mesh exports, get plan-view result figures over an aerial — all in your browser. Nothing is uploaded.

Your figure will appear here once you add the two dataset .h5 files and click Generate figure.
0 Naming convention
Name your SMS simulations starting with EX (Existing) or PR (Proposed) — that's how the app labels each run's condition (and tells the two apart). Without it, titles show the raw simulation name.
1 Data

You need two .h5 files, both exported from your mesh in SMS:

How to export them from SMS

File 1 — mesh geometry

  1. Right-click the mesh (under Mesh Data in the Project Explorer) → Export…
  2. Set Save as type → XMDF File (*.h5), give it a name, and save it anywhere you'll remember.

File 2 — all datasets

  1. Right-click the mesh again → Export Datasets…
  2. In the Select Tree Item dialog, check Z plus the result datasets you want — easiest is to grab them all.
  3. Each simulation has Shear (B_Stress), Froude, Velocity Magnitude, Velocity (x/y), Water Depth, and Water Elevation. Froude isn't needed for these figures.
  4. Give it a name and save.

Drop both files below — any order; the app detects which is which.
